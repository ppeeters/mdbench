# MDBNCH - Molecular Dynamics Benchmark

Translation of the classic Fortran 77 MDBNCH molecular dynamics benchmark to Julia.

## Quick Start

Run the Julia version:
```bash
julia mdbnch.jl
```

## Documentation

See [DOCUMENTATION.md](DOCUMENTATION.md) for detailed information about the translation, algorithm, and usage.

## Contents

- `mdbnch.f` - Original Fortran 77 source code
- `mdbnch.jl` - Julia translation with equivalent functionality
